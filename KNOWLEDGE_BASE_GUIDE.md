## Navigation & Accessibility

### How Article Links Work

All article links use **Next.js Link component** for optimized client-side navigation:

```
/knowledge                    → Knowledge Base Hub (all categories & featured articles)
/knowledge/all               → All articles (sortable, filterable by category)
/knowledge/category/[id]     → Category page (articles grouped by subcategory)
/knowledge/[id]              → Article detail page (full content with sidebar)
```

### Clicking Features

✅ **Fully Functional Click Navigation:**
- Category cards on `/knowledge` → Links to category page
- Featured articles → Direct to article detail page
- Recent articles → Direct to article detail page
- Article links in sidebar → Related articles
- Category filter links → Category pages
- Breadcrumb navigation → Back to parent pages

### Visual Feedback on Hover

All clickable elements have:
- Color transition (text changes to primary color)
- Shadow enhancement on cards
- Arrow icon appears/fades in
- Border color change on focus
- Smooth 200-300ms transitions

## Troubleshooting Article Access

### If Articles Won't Load

1. **Check URL format** - Should be `/knowledge/[article-id]`
2. **Verify article exists** - Check article IDs in guide above
3. **Clear browser cache** - Force refresh with Ctrl+F5
4. **Check network** - Ensure internet connection
5. **Try different browser** - Test in another browser

### If Links Don't Work

1. **Hover over link** - Should show cursor pointer
2. **Check console errors** - Open developer tools (F12)
3. **Verify page exists** - Navigate directly via URL
4. **Test from homepage** - Try navigating from `/knowledge`

## Content Management

### Article Organization

Articles are centrally managed in `/lib/data.tsx` with:
- Unique IDs for routing
- Structured metadata
- Rich HTML content
- Category associations
- Tag systems
- Featured flags

### Adding New Articles

To add a new article:
1. Add entry to `knowledgeArticles` array in `/lib/data.tsx`
2. Provide all required fields (id, title, excerpt, content, etc.)
3. Assign to category (must match existing category ID)
4. Add to `knowledgeCategories` to update article count
5. Deploy to make live
